Database User Setup Script

MySQL Installation & User Creation

sudo apt update
sudo apt install mysql-server

sudo mysql

CREATE USER 'csea'@'localhost' IDENTIFIED BY 'mbccet';
CREATE USER 'cseb'@'localhost' IDENTIFIED BY 'mbccet';
CREATE USER 'ai'@'localhost' IDENTIFIED BY 'mbccet';

GRANT ALL PRIVILEGES ON *.* TO 'csea'@'localhost';
GRANT ALL PRIVILEGES ON *.* TO 'cseb'@'localhost';
GRANT ALL PRIVILEGES ON *.* TO 'ai'@'localhost';

FLUSH PRIVILEGES;
Download MySQL Workbench

For install MySql Work Bench

sudo apt install ./Downloads/mysql-workbench1-community_8.0.45-1ubuntu24.04_amd64.deb

Login with New User

mysql -u csea -p

Oracle Database User Creation

sqlplus / as sysdba

CREATE USER CSEA IDENTIFIED BY tiger;
CREATE USER CSEB IDENTIFIED BY tiger;
CREATE USER AI IDENTIFIED BY tiger;

GRANT create session, create table, create view,
      unlimited tablespace, create procedure TO CSEA;

GRANT create session, create table, create view,
      unlimited tablespace, create procedure TO CSEB;

GRANT create session, create table, create view,
      unlimited tablespace, create procedure TO AI;
